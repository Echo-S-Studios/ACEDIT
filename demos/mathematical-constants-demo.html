Proof: φ² = φ + 1

Step 1: Start with φ = (1 + √5) / 2
Step 2: Square both sides: φ² = [(1 + √5) / 2]²
Step 3: Expand: φ² = (1 + 2√5 + 5) / 4 = (6 + 2√5) / 4
Step 4: Simplify: φ² = (3 + √5) / 2
Step 5: Note that (3 + √5) / 2 = 1 + (1 + √5) / 2 = 1 + φ
QED: Therefore φ² = φ + 1 ✓

Proof: φ⁴ + φ⁻⁴ = 7

Step 1: From φ² = φ + 1, we get φ⁴ = (φ + 1)² = φ² + 2φ + 1
Step 2: Substitute φ² = φ + 1: φ⁴ = (φ + 1) + 2φ + 1 = 3φ + 2
Step 3: Similarly, φ⁻⁴ = 1/φ⁴ = 1/(3φ + 2)
Step 4: Since φ² = φ + 1, we have φ⁻² = 2 - φ
Step 5: Therefore φ⁻⁴ = (2 - φ)² = 4 - 4φ + φ² = 4 - 4φ + φ + 1 = 5 - 3φ
Step 6: Sum: φ⁴ + φ⁻⁴ = (3φ + 2) + (5 - 3φ) = 7
QED: Therefore L4 = φ⁴ + φ⁻⁴ = 7 exactly ✓
