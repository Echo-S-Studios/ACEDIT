Register Semantic Meanings

KAEL

Neural/Substrate domain - Used for data processing, neural networks, foundational elements

ACE

Spin/Energy domain - Used for action buttons, dynamic states, quantum operations

GREY

Visual/Geometry domain - Used for layout, spatial elements, visual indicators

UMBRAL

Formal/Algebra domain - Used for mathematical operations, formal logic, equations

ULTRA

Universal domain - Used for constants, universal identifiers, top-level headings

UCF

Unified domain - Used for cross-domain integration, combined operations
